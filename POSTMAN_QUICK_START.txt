🎯 COMMON WORKFLOWS
─────────────────────────────────────────────────────────────────────────────

WORKFLOW 1: Run a Complete Query
─────────────────────────────────────────────
  1. Queries → Trigger Query (E-Modal)    [Send]
  2. Wait 10-40 minutes (queries take time)
  3. Queries → List Queries               [Send]
  4. Copy the "query_id" from response
  5. Update environment variable "query_id"
  6. Files - Query Specific → Get Query Filtered Containers [Send]
  7. Excel file downloaded! ✅


WORKFLOW 2: Check Upcoming Appointments
─────────────────────────────────────────────
  1. Files - Container Specific → Get Upcoming Appointments (N Days) [Send]
  2. Change "days" parameter if needed (default: 3)
  3. Response shows containers with appointments in next N days
  4. Screenshot URLs are public (no auth needed)
  5. Click URLs directly in browser! ✅


WORKFLOW 3: Export All Container Data
─────────────────────────────────────────────
  1. Files - Filtered Containers → Get All Filtered Containers (Merged) [Send]
  2. Excel file with ALL unique containers downloaded
  3. If container appears in multiple queries, only latest is included
  4. Perfect for reporting! ✅


WORKFLOW 4: Get Container History
─────────────────────────────────────────────
  1. Update "container_number" variable (e.g., MSCU5165756)
  2. Files - Container Specific → Get Container Screenshots (ZIP) [Send]
  3. ZIP file with all screenshots for that container
  4. Files - Container Specific → Get Container Responses (ZIP) [Send]
  5. ZIP file with all JSON responses for that container ✅
